Scala Time

Scala Time is a simple, wrapper for `java.time` time APIs available from JDK versions 1.8 (Java 8) and greater.

Documentation

Usage

This project is not yet considered stable, and the API is subject to change, however snapshot binaries are published to the Sonatype OSS Snapshots public repository.

If using sbt, the current snapshot artefacts can be found at the following coordinates:
JDK 1.8

"codes.reactive" %% "scala-time" % "0.3.0-SNAPSHOT"

JDK 1.7

"codes.reactive" %% "scala-time-threeten" % "0.3.0-SNAPSHOT"


See version specific documentation for usage examples and information.
